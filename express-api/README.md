# Express API Server

External API server for Next.js 15 tutorial.

## Run the server

\`\`\`bash
npm run dev
\`\`\`

Server will run on http://localhost:3001

## API Endpoints

- GET /api/health
- GET /api/products
- GET /api/products/:id
- GET /api/products/:id/reviews
- POST /api/products/:id/reviews
- GET /api/users
- GET /api/users/:id
